#### A) Grammar (CORE: up to 6)

##### Grammar Point: `Partitiivi: -a/-ä (perus)`
- **Rule (EN)**: Use the partitive case to mark an object especially in (1) negative sentences, and (2) with many common “partitive verbs” where the object is not seen as a completed whole. In this lesson you see the basic ending **-a/-ä** after many stems (e.g., *talo → taloa*). Formation (as shown): add **-a/-ä** according to vowel harmony. **Note**: the lesson states that in the partitive there is **no k-p-t consonant gradation** in the ending choice rule it presents.
- **Forms/patterns**: `a/ä: taloa, bussia, kylmää, valoa; Huom! Partitivissa ei k-p-t-vaihtelua.`
- **Examples (FI)**:
  - Minä rakastan häntä.
  - Minä etsin työpaikkaa.
  - Me katsomme televisiota.
  - Ymmärrätkö englantia?
- **Speaking prompt**: Mitä sinä etsit?
- **Writing frame**: Minä etsin ____.
- **Common error warning**: In partitive, don’t add consonant gradation in the ending pattern shown here. **(lesson)**
- **Validation check**:

  - **Notes (EN)**: Verified that partitive is a key object case (esp. negation / incomplete or non-total object contexts) and that object case alternation is a standard rule in Finnish.

##### Grammar Point: `Partitiivi: verbien rektio (auttaa/etsiä/katsoa/puhua/rakastaa/ymmärtää)`
- **Rule (EN)**: Some verbs require their object in the partitive by rection (rektio). This is not “logic”; it’s a fixed pattern you learn with each verb. In this lesson, verbs like *auttaa, etsiä, katsoa, puhua (kieli), rakastaa, ymmärtää* take a partitive object. Formation: choose the correct partitive form of the object word (as shown in lesson examples).
- **Forms/patterns**: `auttaa + P; etsiä + P; katsoa + P; puhua + P (kieli); rakastaa + P; ymmärtää + P`
- **Examples (FI)**:
  - Voitko auttaa minua?
  - Olga etsii työpaikkaa.
  - Me katsomme televisiota.
  - Julia puhuu saksaa.
- **Speaking prompt**: Voitko auttaa minua?
- **Writing frame**: Voitko auttaa minua ____?
- **Common error warning**: Don’t use a “total object” form with these verbs; use partitive as shown. **(lesson)**
- **Validation check**:
  - **Status**: `ok`
  - **Notes (EN)**: Confirmed these are commonly taught as partitive-verb rections; rection is a recognized concept in Finnish grammar guidance.
  - **Sources**:

    - https://kielitoimistonohjepankki.fi/ohje/rektioita-mita-rektio-on/

##### Grammar Point: `Kieltolause + partitiivi (ei …)`
- **Rule (EN)**: In negative clauses, the object is in the partitive. In this lesson: *Minulla ei ole autoa.* This rule is very high-impact for writing and speaking because “ei” is frequent. Formation: use **ei** + verb, and put the object in the partitive (as shown).
- **Forms/patterns**: `Minulla ei ole + partitiivi (autoa, kissaa, miestä)`
- **Examples (FI)**:
  - Minulla ei ole autoa.
  - Pedrolla ei ole kissaa.
  - Päivi Koskelalla ei ole miestä.
  - Minulla ei ole töitä Suomessa vielä.
- **Speaking prompt**: Onko sinulla auto?
- **Writing frame**: Minulla ei ole ____.
- **Common error warning**: Don’t keep the object in nominative after **ei**; switch to partitive. **(general)**
- **Validation check**:
  - **Status**: `ok`
  - **Notes (EN)**: Verified the core rule: negation triggers partitive object in Finnish.
  - **Sources**:
    - https://kielitoimistonohjepankki.fi/ohje/objekti-han-luki-kirjaa-kirjan-milloin-partitiivi-milloin-ei/

##### Grammar Point: `Numero + partitiivi`
- **Rule (EN)**: When a numeral is in basic form (like 6, 9, 200), the following noun is typically in the partitive singular in Finnish. This lesson shows clear examples: *6 jäätelöä, 9 pitkää miestä, 3 hyviä tietokonetta*. Formation: numeral + noun in partitive; the adjective also matches the partitive form as shown in the examples.
- **Forms/patterns**: `6 jäätelöä; 9 pitkää miestä; 3 hyviä tietokonetta`
- **Examples (FI)**:
  - Meillä on yksi lapsi.
  - 6 jäätelöä.
  - 9 pitkää miestä.
  - 3 hyviä tietokonetta.
- **Speaking prompt**: Kuinka monta lasta?
- **Writing frame**: Minulla on ____.
- **Common error warning**: After a numeral (2+), don’t use plain nominative; use the partitive pattern shown. **(general)**
- **Validation check**:
  - **Status**: `ok`
  - **Notes (EN)**: Verified that numerals in basic form typically take a partitive noun (e.g., “200 autoa” type guidance).
  - **Sources**:
    - https://kielitoimistonohjepankki.fi/asiasana/numeraalit/

##### Grammar Point: `Genetiivi + kanssa`
- **Rule (EN)**: With the postposition **kanssa** (“with”), the word before it is typically in the genitive (ends in **-n**). This is because Finnish postpositions usually take a genitive complement. Formation: X + **-n** + kanssa (as shown).
- **Forms/patterns**: `Pedron ja Alexin kanssa; Hannan kanssa; opettajan kanssa`
- **Examples (FI)**:
  - Olen samalla suomen kurssilla Pedron ja Alexin kanssa.
  - Pedro asuu Hannan kanssa.
  - Hän puhuu opettajan kanssa.
- **Speaking prompt**: Kenen kanssa sinä olet samalla suomen kurssilla?
- **Writing frame**: Olen ____ kanssa.
- **Common error warning**: Don’t use nominative before **kanssa** in standard Finnish; use genitive. **(general)**
- **Validation check**:
  - **Status**: `ok`
  - **Notes (EN)**: Verified that postpositions in Finnish typically require a genitive complement; this covers “-n kanssa”.
  - **Sources**:
    - https://kielitoimistonohjepankki.fi/ohje/post-ja-prepositiot/
    - https://kielitoimistonohjepankki.fi/ohje/post-ja-prepositiot-mista-niita-tulee/

##### Grammar Point: `Pitää + -sta/-stä (mistä?)`
- **Rule (EN)**: In this lesson, liking is expressed with **pitää jostakin**. The complement answers “mistä?” and uses **-sta/-stä** (elative). Formation: *pitää* + noun + **-sta/-stä** as shown.
- **Forms/patterns**: `pitää + sta/stä; Mistä sinä pidät?`
- **Examples (FI)**:
  - Mistä sinä pidät?
  - Minä pidän vanilijajäätelöstä.
  - En pidä mansikkajäätelöstä.
  - Minä pidän asunnostani.
- **Speaking prompt**: Mistä sinä pidät?
- **Writing frame**: Minä pidän ____sta/____stä.
- **Common error warning**: Don’t use nominative after *pitää*; use **-sta/-stä** as shown. **(lesson)**

#### D) Speaking Training (record yourself)

##### 1) Monologue (60–90 seconds)
- **Prompt (FI)**: Kerro itsestäsi: perheestä, asunnosta ja siitä, mistä sinä pidät. Kerro myös, oletko töissä vai opiskelija, ja missä asut.
- **Plan (5 bullets)**:
  - Minun nimi on ____ ja olen ____ vuotta vanha.
  - Minulla on / Minulla ei ole ____.
  - Me asumme ____ssä, mutta ____.
  - Minä pidän ____stä, koska ____.
  - Nyt olen ____ ja sitten ____.
- **Must-use connectors**: koska, mutta, sitten, myös

##### 2) Role-play (8–12 lines)
- **Situation (EN)**: You are in a clothing store. You want a jacket, ask size/color, ask where the fitting room is, and buy it.
- **Dialogue**:
  - **B**: Hei, voinko auttaa?
  - **A**: Kiitos, mä vaan katselen.
  - **A**: Mä etsin farkkutakkia.
  - **B**: Hetki, mä tarkistan... Joo, on. Tässä, ole hyvä.
  - **A**: Kiitti. Entä onks tätä myös mustana?
  - **B**: Ei valitettavasti.
  - **A**: Okei. Tota, missä sovituskoppi on?
  - **B**: Se on tuolla takana.
  - **A**: Mä ottaisin tän takin.
  - **B**: Tarvitsetko muovipussin?
  - **A**: Ei kiitos, mulla on oma kassi.

##### 3) Q&A (6)
1. **Q**: Kuka sinä olet?

   **A (model)**: Minulla ei ole perhettä Suomessa.
4. **Q**: Millainen sinä olet?

   **A (model)**: Minä pidän asunnostani, koska se on valoisa ja viihtyisä.

#### F) Drills (fast practice)

##### 1) Substitution drill (8)
1) Minulla on kiva asunto. → Minulla on iso suku.  
2) Minulla ei ole autoa. → Minulla ei ole töitä Suomessa vielä.  
3) Me asumme Helsingissä. → Minä asun Helsingissä.  
4) Olen rauhallinen. → Olen iloinen.  
5) Hän on pitkä ja hoikka. → Hän on söpö 4-vuotias tyttö.  
6) Minä etsin työpaikkaa. → Mä etsin farkkutakkia.  
7) Minä pidän asunnostani. → Minä pidän takista.  
8) Mä etsin farkkutakkia. → Mä etsin villapuseroa.

##### 2) Transform drill (6)
1) (positive → negative) Minulla on auto. → Minulla ei ole autoa.  
2) (statement → question) Minulla on auto. → Onko sinulla auto?  
3) (question → short answer) Onko sinulla auto? → (Joo), on.  
4) (standard → spoken) Minulla on oma kassi. → Mulla on oma kassi.  
5) (standard → spoken) Minulla ei ole autoa. → Mulla ei oo autoa.  
6) (standard → spoken) Minä rakastan sinua. → Mä rakastan sua.

##### 3) Case/object drill (6)
1) rakastaa + P: Minä rakastan ____ (hän). → Minä rakastan häntä.  
2) etsiä + P: Minä etsin ____ (työpaikka). → Minä etsin työpaikkaa.  
3) auttaa + P: Voitko auttaa ____ (minä). → Voitko auttaa minua?  
4) katsoa + P: Me katsomme ____ (televisio). → Me katsomme televisiota.  
5) puhua + P: Julia puhuu ____ (saksa). → Julia puhuu saksaa.  
6) Minulla ei ole + P: Minulla ei ole ____ (auto). → Minulla ei ole autoa.
